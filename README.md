# Kegiatan Dosen & Send Wa


## Installation & Execution

### Laravel
```
git clone https://github.com/defrindr/2024-penjadwalan-dosen.git
cd 2024-penjadwalan-dosen
composer install
php artisan migrate:fresh --seed
php artisan serve
```
### Whatsapp Instance
```
cd whatsappp-node
npm install
npm run start
```
